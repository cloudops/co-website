CI/CD

During this workshop you will learn the tools and best practices to build, test, and ship software as well as enhance and automate your delivery pipeline.

Duration: 1 day

$1500/attendee (minimum 6 attendees)

Prerequisites:

Experience using Linux CLI, good understanding of Source Code Management (SCM) systems such as Git, experience using Docker and Kubernetes, a general understanding of CI/CD process is a plus

Technical Requirements:

Mac, Linux OS or Windows laptop with SSH client (putty, cygwin), web browser supporting HTML5

Course Outline

  • Overview – The State of DevOps
  • Concourse CI
  • CircleCI
  • GitLab CI
  • Jenkins Pipeline
  • Drone
  • Fabric8
  • Trying It All Together – Pipelines and
    Orchestrators
