Docker and Kubernetes

CloudOps is a cloud consulting, managed services, and software company focused on open source, cloud native platforms, networking and DevOps

Au cours de cet atelier, vous apprendrez:

  1. Comment Hashicorp Terraform de Hashicorp est un outil d’infrastructure multi-plateformes qui vous permet d’utiliser le même outillage sur plusieurs plateformes afin d’automatiser l’infrastructure;
  2. Comment Ansible peut être utilisé pour gérer les configurations d’applications distantes et les installations de machines virtuelles bâties sur votre déploiement de l’infrastructure Terraform.

Durée: 1 jour

Sur place: 1500 $ / participant (minimum de 6 participants)

Prérequis:

Connaissance de base des concepts et architectures de l’infonuagique

Exigences techniques:

Ordinateur portable Mac, Linux OS ou Windows avec client SSH (putty, cygwin), navigateur Web compatible avec HTML5. Installations de Terraform, de Ansible and ainsi que du module d’extension terraform-provider-cloudca

  • – Installer et configurer Terraform
