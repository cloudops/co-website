OpenShift

During this workshop you will learn how to install, administer, and use Red Hat OpenShift container platform to deploy your enterprise applications.

Durée: 2 jours

$2500/participant (minimum 6 participants)

Conditions préalables:

Experience using Linux CLI and a general understanding of virtualization and container technology

Exigences techniques:

Mac, Linux OS or Windows laptop with SSH client (putty, cygwin), web browser supporting HTML5

Plan de cours

Jour 1
  • Introduction aux microservices
  • Docker Basics
  • Advanced Docker
  • Managing Docker
  • Docker Tooling
  • Docker Ecosystem
Jour 2
  • OpenShift cluster features, architecture, and sizing
  • Developer Introduction to OpenShift
  • Investigate OpenShift cluster installation methods
  • OpenShift Container Platform general configuration and management
  • Configure storage providers and storage classes.
  • Deploy applications on OpenShift Platform Container using Source-to-Image (S2I)
  • Networking
  • Diagnose cluster health
  • Users and Policies management
  • Scale OpenShift clusters
  • Manage OpenShift resources
