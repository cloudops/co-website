DevOps Workshops

Microservices, Docker & Basic Kubernetes (3 days)

CloudOps’ flagship workshop covers microservices architecture principles and the mechanics of container tooling; now a key model for application development.

Advanced Containers & Kubernetes (2 days)

In this workshop, explore app and infrastructure monitoring tools (open source and other), their configurations, application performance monitoring, and altering and visualizing data.

Secrets Management (1/2 Day)

This workshop covers Hashicorp Vault as a reliable tool for secrets management and data protection. It also incorporates operational components and techniques for integrating Vault secrets into applications.

Infrastructure as Code (1 day)

Reduce errors and technical debt: automate! In this workshop, learn how Hashicorp Terraform and Ansible work as a cross platform, infrastructure tools.

Kubernetes and Machine Learning (2 days)

If you do analysis and automated analytical model building, this workshop is for you. Learn how to run deployments of ML workflows on Kubernetes to make them simple, portable, and scalable.

OpenShift (2 days)

OpenShift simplifies Kubernetes management when running a large number of apps at scale. Learn about its built-in automation, how it increases efficiency in app cycles and reduces infrastructure costs.

CI/CD (1 day)

Need a place to start for CI/CD and DevOps? This one day course will walk through the tools and best practices to build, test, and ship software and automate your delivery pipeline.
