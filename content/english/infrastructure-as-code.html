Docker and Kubernetes

CloudOps is a cloud consulting, managed services, and software company focused on open source, cloud native platforms, networking and DevOps

During this workshop you will learn:

  1. How Hashicorp Terraform by Hashicorp works as a cross platform, infrastructure tool enabling you to use the same tooling over multiple platforms for infrastructure automation;
  2. How Ansible can be used to manage remote application configurations and VM installations built on top of your Terraform infrastructure deployment.

Duration: 1 day

On Premise: $1500/attendee (minimum 6 attendees)

Prerequisites:

Basic knowledge of cloud computing concepts and architectures

Technical Requirements:

Mac, Linux OS or Windows laptop with SSH client (putty, cygwin), web browser supporting HTML5. Installations of: Terraform, Ansible and the cloud.ca Terraform provider plugin
